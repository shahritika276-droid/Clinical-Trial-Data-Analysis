* Clinical-Trial-Data-Analysis
Exploratory data analysis of clinical trial data using Python.
* Clinical Trial Data Analysis

* Project Overview
This project performs exploratory data analysis on clinical trial data to understand
trial phases, statuses, and enrollment trends using Python.

* Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

* Key Analysis
- Distribution of clinical trial phases
- Trial status analysis
- Phase vs Status heatmap
- Enrollment analysis by phase and status

* Outcome
The project highlights how clinical trials progress across phases and how
participant enrollment varies with trial status.
